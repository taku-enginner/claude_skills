# Flutter実装フロー

実装内容: $ARGUMENTS

## 自動実行フロー

以下のフローを自動で実行してください:

### 1. 実装
- $ARGUMENTS の要求に基づいてDartコードを実装
- 適切なディレクトリに配置（lib/配下）

### 2. テストケース生成
```bash
python .claude_skills/code/generate_tests.py <実装したファイル> --force
```

### 3. テスト実行
```bash
python .claude_skills/code/run_tests.py <テストファイル>
```

### 4. 結果に応じた対応
- **成功**: 完了報告
- **失敗（1-3回目）**: 実装を修正して再テスト
- **失敗（3回）**: ユーザーに相談

### 完了報告フォーマット
```
## 実装完了

### 実装内容
- [実装した機能]

### ファイル
- lib/xxx.dart
- test/xxx_test.dart

### テスト結果
- 成功: X件 ✅
```

### 3回失敗時
```
## テストが3回失敗しました

### 失敗しているテスト
- [テスト名]: [エラー]

### 試した修正
1. [修正1]
2. [修正2]
3. [修正3]

どのように対応しますか？
```
